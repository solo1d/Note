
压缩包管理:    
    压缩:   tar/rar/zip   参数 压缩包名 原材料
    解压缩: tar/rar/unzip 参数 压缩包名 解压缩路径
                     rar 解压缩到指定目录不需要指定参数
                     unzip 不需要解压参数,但是指定路径需要参数 -d .


    常见压缩格式有:
        gz  -- 使用gzip 命令 (指定文件后缀变成.gz 或者 .tar.gz)
        bz2 -- 使用bzip2命令 (指定文件后缀变成.bz2 或者 .tar.bz2)
        rar -- 使用rar  命令 (需要单独安装 rar 这个工具,后缀 .rar)
        zip -- 

        常见压缩命令:
            tar  -  打包(这个命令实际是间接调用 gzip 和 bzip2 命令来包装实现的)
                参数 :
                    c - 创建压缩文件
                    x - 释放压缩文件
                    v - 打印提示信息
                    f - 指定压缩包的名字(解压缩和进行压缩, 要有规范)
                    z - 使用 gzip  的方式压缩和解压文件 (文件名后戳一般是 xxx.tar.gz) 
                    j - 使用 bzip2 的方式压缩和解压文件 (文件名后戳一般是 xxx.tar.bz2)
                压缩范例:
                    tar 参数 压缩包的名字 源文件 多个源文件 更多的源文件    (gz格式)
                        
                        tar zcvf "new.tar.gz" "file1" "file2" "dir1" 
                        // 会生成一个压缩包, 名字是 "new.tar.gz" . 里面存在三个压缩内容.
                解压缩范例:
                        tar 参数 压缩包名字  -C 解压到的目录可写可不写
                        tar zxvf "new.tar.gz" "new"
                        // 解压到当前目录的"new"文件夹内.解压的时候会覆盖同名文件,要小心.

                        tar zxvf "new.tar.gz" -C ~/cpp/  
                        // 解压到 cpp 目录下,  -C 这个参数是指定目录的.



                rar  -  打包 (需要单独安装,而且只针对rar 这一种压缩包)
                        参数:
                                a - 创建压缩文件
                                x - 释放压缩文件
                                r - 递归进行(压缩文件夹时需要添加)

                        压缩范例:
                rar a "new.rar" "file1" "file2" -r 
                // 生成一个压缩包 new.rar , 内含 file1 和 file2 这两个文件.
            解压缩范例:
                rar x  "new.rar"  /home/pi
                // 解压缩包 new.rar , 解压到指定目录, 如果不指定就会解压到当前目录.


        命令 zip/unzip: (只针对 .zip 格式的压缩文件, 需要额外安装)

        zip  参数  压缩包名  原材料 
            如果有目录就添加参数  -r  ,否则无参数.

        unzip  压缩包名字 -d 指定解压目录
            如果不指定目录, 那么也不需要添加 -d 这个参数
