运行环境为 ubuntu  
环境变量PATH 就是路径名,路径里面存放着运行库和命令之类的路径,命令运行前会首先寻找路径.

命令   file 文件名或程序名.  可以拿到程序和文件的属性.
命令   ldd  程序名          可以得到这个程序运行的时候需要哪些运行库.(后面有地址的代表加载到了,没有就有问题)

gcc 生成的文件都是 ELF 格式的可执行文件, 是由 ld-linux.os* (动态加载器) 来完成的.



man 手册文档命令需要章节号, 方便快速定位.
        1 可执行程序或shell命令的手册.    man 1 ls ;
        2 系统调用(内核提供的函数).       man 2 open ;
        3 库函数调用.    man 3  printf ;
        5 文件格式和规范, 如 /etc/passwd  .   man /ect/passwd


快捷键记录(每次都使用CTRL+): 上p,下n,左b,右f,删除前一个字符h,删除后一个字符d,前面所有u,后面所有k.

执行ls 命令弹出带颜色的目录和文件详解:
        白色: 普通文件
        绿色: 可执行文件
        红色: 压缩文件
        蓝色: 目录
        青色: 软链接文件    就是快捷方式
        黄色: 设备文件      块设备,字符设备,等等
        灰色: 其他文件

在命令参数前的-(横杠)   一般字符的话只需要一个横杠(ls -a), 如果参数是单词的话需要两个横杠(ls --help).


mkdir  -p  这个参数是递归创建文件夹.

命令解析器的作用(虚拟终端工作原理)
    向虚拟终端输入命令; shell 解析你输入的命令,然后搜索环境变量:路径 , 找到输入的命令,运行这个程序,终端有对应输出.
 

Linux根目录下常见目录的功能已经里面存放的内容:
    bin  :binary, 二进制文件,可执行程序,一般要用到的shell命令.
        dev  :device, 设备文件.
        lib  :library,linux运行的时候需要加载的一些动态库.
        mnt  :mount,  手动的挂载目录.
        media:        外设的自动挂载目录.
        root :        linux超级用户的家目录.
        usr  :unix system resoure, linux/unix系统的资源目录.(有头文件include,用户安装的应用程序local).
        etc  :and so on, 存放配置文件.(无论是操作系统还是安装的软件对应的配置文件都会放到etc下面).
        opt  :        安装第三方应用程序.
        home :        操作系统所有用户的家目录(系统目录/home).(用户家目录是/home/pi, 也说成是宿主目录).
        tmp  :temp,   存放临时文件,重启后清空该目录.
        boot :        启动目录,存放开机启动项,没事别乱动.



相对路径: 从当前目录开始表示.
绝对路径: 从根目录开始表示.

GUI   图形用户接口.


命令 rm /home/pi/temp/* -r    //这个表示删除temp 这个目录内的所有文件


命令 tree  [参数是目录名]   
    作用是 查看目录文件结构 (需要单独安装 apt-get install tree).
