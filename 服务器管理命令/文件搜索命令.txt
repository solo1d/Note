文件搜索命令	$find /etc -name init          //区分大小写
find      [所搜范围] [-搜索属性选项] [匹配条件]      //精准搜索
          匹配条件???   //搜索开头相同     
         *匹配条件*     //模糊搜索
         文件名搜索(-name)          //区分大小写  // (-type)
         文件名搜索(-iname)         //不区分大小写

         文件大小搜索(-size) (+大于查找值) (-小于)   //查找单位是数据块
   $find /etc -size +1638 -a -size -2048      // (-a)同时满足
         在etc下查找大于8mb 小于100mb的文件   // (-o)其中一个

         根据所属组查找(-user)    //所属组
         时间属性搜索: -amin      //访问时间   (-5) 五分钟之内
                       -cmin      //文件属性   (+5)已超过5分钟的更改
                       -mmin      //文件内容  
 
        文件类型查找 (-type)         //加在文件查找后面

       $find . -name inittab -exec ls -l {} \;   //注意空格
 . 是当前目录|     在当前目录下查找inittab文件并显示其详细信息
        -exec 或者 -ok执行后续的操作
        {} 可插入查询结果
         \ 转义符
         ; 结束

         根据 i 节点查找(-inum)          //使用 ls -i 可以找到 i 节点

在文件资料库中查找文件            
locate        [文件名或目录]
                $locate inittab           //查找 inittab
mocate      同等

-----------------------------------------------------------------------------------
查找命令所在的目录
which       [命令]
              $which rm
                alias rm='rm -i'                /usr/sbin/rm
               // rm是 rm -i 的别名            -i 是确认的意思
              一般命令都在 bin/usr/bin下面, sbin/usr/sbin 是 root使用的命令
-----------------------------------------------------------------------------------
查找命令所在目录和帮助文档
whereis	[命令]
              $whereis rm
                rm: /usr/bin/rm  /usrshare/man/man1/rm.1.gz
                命令所在的路径   ,  命令帮助文件所在的目录
-----------------------------------------------------------------------------------
从文件中搜索字符串匹配行并输出
grep       [选项] [指定字串] [文件]
-i            不区分大小写
-v            指定字符串
-n            输出行号
--color=auto       搜索出关键字用颜色显示

            $grep -i --color=auto "mysql"  /root/install.log
              在 install 文件内容中查找 mysql 不区分大小写,并用颜色显示
            $grep -v ^# /root/install.log
              显示 install 文件内容并且排除以 # 开头的行数 ( ^ 是提示开头)
