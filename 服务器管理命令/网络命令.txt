vim /etc/sysconfig/network-scripts/ifcfg-ens33
 把文件中的 ONBOOT=no 改成 yes 就可启用网卡
该文件就是网络配置文件

网络yum源配置文件所在目录
/etc/yum.repos.d/CentOS-Base.repo
详细信息在 "软件安装管理命令 " 中
-----------------------------------------------------------------------------------
查看和设置网卡信息               (只有root 可以使用)
ifconfig	     [ 网卡(修改时添加 )] [网络地址 (修改时添加) ]
                     +IP地址 直接服务器地址
                      网卡:num  给网卡添加另一个地址(num虚拟卡号)
                    $ifconfig ens33 192.168.1.111
                      给网卡 ens33 分配ip地址   (lo 是回环网卡)
-----------------------------------------------------------------------------------
测试网络连通性
ping            [选项] [IP地址]
-c                指定发送次数
                 $ping 192.168.1.111

-----------------------------------------------------------------------------------
给一个在线的用户发信息          (用户在线才可以发送) 
write              <用户名>
                     写完后 ctrl + D 结束输入发送

-----------------------------------------------------------------------------------
给所有在线用户发送命令            ( 等同于广播 )
wall              [发送的信息]

-----------------------------------------------------------------------------------
查看发送电子邮件           (在邮箱内输入 help 可以获得帮助)
mail              [ 用户名(发送邮件时添加) ]
                     添加用户时是发送邮件, 不添加用户时是进入自己的邮箱

-----------------------------------------------------------------------------------
列出目前与过去登入系统的用户信息         (!重要查询命令)
last                 可以判断系统是否重启过
lastlog            可以查看所有用户的最后登录时间和ip 以及端口
lastlog -u [用户UID]      只查看这个用户的登录信息

-----------------------------------------------------------------------------------
显示数据包到主机间的路径

traceroute          [ 网站的地址 ]
-----------------------------------------------------------------------------------
显示网络状态相关信息
netstat                [ 选项 ]     (可以一次输入多个选项)
-t                  TCP协议
-u                 UDP协议
-l                  查看监听的端口  
-r                  查看路由,也就是查看网端
-n                 显示IP地址和端口号

主要使用的三种组合
netstat  -tlun               查看本机已开启的监听端口
netstat  -an                 查看本机所有的网络连接
netstat  -rn                 查看本机路由表
-----------------------------------------------------------------------------------
进入网络管理器                (setup   在centos7已废弃)
nmtui                    可以编辑连接,启动连接,设置系统主机名
service network restart          重启网络服务// 必须重启才能生效
