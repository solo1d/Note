-----------------------------------------------------------------------------------
常用文件操作指令
ls 	可不带任何选项
-a	显示所有项目(包括隐藏文件
-d /	显示目录属性
-l	显示详细信息
-i	i 节点
-lh	把详细信息的字节数转换成kb,mb,gb
-----------------------------------------------------------------------------------
创建目录
mkdir	[目录][需要创建的目录] /tmp/123 创建的是123	目录
-p	递归创建,多层创建
-----------------------------------------------------------------------------------
创建文件
touch	[目录和文件名](使用双引号可以创建带空格的文件
-----------------------------------------------------------------------------------
删除目录(只能删除空目录)
rmdir	[目录]
-----------------------------------------------------------------------------------
删除文件和目录(包括所有文件)
rm	[文件或者目录]
-r	删除目录
-f	强制删除(不需要确认)&一定要小心
             $rm -rf * 
                删除该目录下所有文件
-----------------------------------------------------------------------------------
切换目录
cd	[目录]
..	退回当前目录的上一层目录
-----------------------------------------------------------------------------------
查看所在目录的位置
pwd	显示当前所在的目录
-----------------------------------------------------------------------------------
复制-还可以改名
cp	[复制文件目录][目标目录(或修改文件名)]
-r	复制目录
-p	保留文件属性,比如最后修改时间
-----------------------------------------------------------------------------------
剪切文件,改名
mv	[源文件目录][目标目录]
	(原地剪切就是改名)

***********************************************************************************
查看小文件内容(小文件,短文件)
cat	[文件目录和文件名]
-n	显示行号
-----------------------------------------------------------------------------------
查看大文件内容(大文件,长文件)
more	[目录和文件名]	
	(空格或f)	翻页
	(Enter)	换行
	(q或Q)	退出
	不可向上翻页
-----------------------------------------------------------------------------------
查看大文件内容(可以向上翻)
less	+目录和文件名
	(pu up)	向上
	(pg dn)	向下
	(上下箭头)	   翻行
	(/关键词,按n可以向下找)    搜索
-----------------------------------------------------------------------------------
动态显示文件末尾内容(监控日志)
tail	[文件目录]
-n	指定显示的行数
-f	动态显示末尾(实时更新)
-----------------------------------------------------------------------------------
生成链接文件	(.soft 软连接文件后缀)(.hard 硬)
ln	[源文件][目标文件.soft]
	带参数就是软连接,不带参数就是硬链接
-s	创建软连接
 
***********************************************************************************
压缩成 gz 文件     (只能压缩文件)
gzip          [选项 (可不加 )][文件]  (压缩之后是 .gz 文件,且源文件删除)
-d              解压缩

-----------------------------------------------------------------------------------
解压 gz 的压缩文件
gunzip      [压缩文件 .gz ]

-----------------------------------------------------------------------------------
压缩目录和文件或已经压缩的文件进行打包     (不会删除原文件和目录)
tar           [选项] [压缩后的文件名] [目录]      (压缩后的格式 .tar.gz)
-c            打包                                         
-v            显示打包过程中的详细信息      (永远写在f前选项最后段)
-f            指定打包后的文件名 [文件名写在即将被压缩目录的前面] (写在选项最后)
-z            打包同时压缩,如果对.tar.gz压缩包使用 则进行解压缩
-x            解包
-j            压缩或解压  bz2 格式文件
           $tar -cvf aaa.tar.gz  /usr/sss                         (选项顺序非常重要)
              将sss目录打包成 aaa.tar.gz 并显示过程
            $tar  xzvf   aaa.tar.gz              (互换z和c,并添加了x )
              解aaa压缩,放到当前目录
-----------------------------------------------------------------------------------
压缩文件或目录 ( zip 文件压缩) (可保留源文件) (不可以和 .gz 一起使用)
zip          [选项] [压缩后文件名] [文件或目录]
-r            [压缩目录]

解压缩 ( zip 文件 ) (不可以和 .gz 一起使用)
unzip        [压缩文件]

^^^^^^^^^^^^^^^
升级版压缩文件和目录 ( bz2  2 文 件压缩) (不保留源文件) (不可以和 .gz 一起使用)
bzip2        [ 选项  ] [文件或目录名]
-k              保留源文件
              $tar -cjf aaa.tar.bz2 aas
                和tar一起使用 使用 j 可以生成 bz2 压缩文件

解压缩 bz2 文件       (不保留源文件)  (不可以和 .gz 一起使用)
bunzip2       [ 选项 ] [压缩文件名]
-k                 保留源文件
                 $bunzip2 -k aaa.bz2
                    解压aaa.bz2 压缩文件
                 $tar -xjf aaa.tar.bz2 
                    解压 aaa.tar.bz2 文件

***********************************************************************************
挂载命令          (需要先创建一个挂载点空二级目录)     (目录 /mnt为挂载目录 )
mount          -t  [ 需要挂载的文件类型 ] [ 已创建的挂载目录 ]
                          iso9660 光盘文件类型
                       $mount -t ios9660 /dev/sr0 /mnt/cdrom/
                         将光盘挂载到mnt/cdrom目录下

卸载命令
umount          [ 挂载目录 ]
